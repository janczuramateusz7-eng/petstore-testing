## PS-0006 — Password field is not empty on the Sign In page

**Module:** Sign In  
**Severity:** Major  
**Priority:** Medium  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is on the Home page.

### Steps to Reproduce
1. Open the website.
2. Click the [Sign In] link.

### Expected Result
The Password field should be empty when the Sign In page is opened.

### Actual Result
The Password field is pre-filled (dots are visible), indicating that a password value is already present.
