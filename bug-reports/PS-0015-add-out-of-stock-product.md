## PS-0015 — User can add out-of-stock products to the cart

**Module:** Cart / Business Logic  
**Severity:** Major  
**Priority:** High  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is logged in.

### Steps to Reproduce
1. Open the website.
2. Log in.
3. Find a product that is out of stock.
4. Attempt to add it to the cart.

### Expected Result
The system prevents adding out-of-stock products.

### Actual Result
The user can add out-of-stock products without restriction.
