## PS-0010 — Product name is not displayed in the cart after adding a product

**Module:** Cart  
**Severity:** Major  
**Priority:** High  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is logged in.

### Steps to Reproduce
1. Open the website.
2. Log in with valid credentials.
3. Add a product to the cart.
4. Open the Cart page.

### Expected Result
The product name is displayed in the cart.

### Actual Result
The product name is not displayed in the cart.
