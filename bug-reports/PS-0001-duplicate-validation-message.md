## PS-0001 — Duplicate validation messages are displayed after clicking the empty Search field

**Module:** Home Page / Search  
**Severity:** Minor  
**Priority:** Medium  
**Environment:** PetStore web application, Desktop (Chrome)

### Preconditions
User is on the Home page.

### Steps to Reproduce
1. Open the Home page.
2. Click on the [Search] field.
3. Leave the field empty.
4. Click the [Search] button.

### Expected Result
Only one validation message should be displayed informing the user that the Search field is required.

### Actual Result
Two identical validation messages are displayed for the empty Search field.
