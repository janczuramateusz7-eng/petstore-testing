## PS-0007 — Category order is inconsistent between top navigation and sidebar

**Module:** Home Page / Navigation  
**Severity:** Minor  
**Priority:** Low  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is on the Home page.

### Steps to Reproduce
1. Open the website.
2. Observe the category order in the top navigation bar.
3. Compare it with the category order in the left sidebar.

### Expected Result
The category order should be consistent across all navigation sections.

### Actual Result
The category order differs between the top navigation bar and the left sidebar.
