## PS-0013 — No validation message is displayed when exceeding product quantity limit

**Module:** Cart / Validation  
**Severity:** Major  
**Priority:** High  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is logged in.

### Steps to Reproduce
1. Open the website.
2. Log in.
3. Add more than 5 identical products to the cart.

### Expected Result
A validation message is displayed informing the user about the quantity limit.

### Actual Result
No validation message is displayed.
