## PS-0009 — Cart icon does not display a tooltip labeled "Cart"

**Module:** Cart / UI  
**Severity:** Minor  
**Priority:** Low  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is on any page of the application.

### Steps to Reproduce
1. Open the website.
2. Hover over the Cart icon.

### Expected Result
A tooltip labeled "Cart" is displayed.

### Actual Result
No tooltip is displayed when hovering over the Cart icon.
