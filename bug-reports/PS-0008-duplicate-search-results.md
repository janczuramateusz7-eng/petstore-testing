## PS-0008 — Duplicate results are displayed when searching using a two-word query

**Module:** Search  
**Severity:** Minor  
**Priority:** Low  
**Environment:** Windows 11, Chrome (latest), PetStore Demo

### Preconditions
User is on the Home page.

### Steps to Reproduce
1. Open the website.
2. Enter a two-word query (e.g. "Golden Retriever") in the Search field.
3. Click the [Search] button.

### Expected Result
Each search result should be displayed only once.

### Actual Result
Each unique search result is displayed twice.
